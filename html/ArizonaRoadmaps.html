Arizona Roadmaps

A mobile app developed with the Flutter Framework

Arizona Roadmaps focuses on campgrounds native to Arizona. The vision is to become an app that will help users explore the National Forests of Arizona. A beautifiul and heavily underrated state; Arizona has a lot to offer those that wish to adventure. The app focuses on providing a list of hazards one's vehicle should triumph if exploring the state parks.

Motivation and Goals

Safe Adventures

AZ Roadmaps strives to provide the amateur Arizona camper with all the necessary requirements any vehicle should pass if off-roading. Each Campsite has been verified for possible hazards that could affect any vehicle. Vehicle Reccomendations are provided so that adventurers know which vehicle precautions are reccomended for the journey.

Modern Campsites

Campgrounds have been popping up all over Arizona. AZ Roadmaps provides a great interactive way to see the popular Arizona Campsites. Users can fill a form to request the addition of a new campsite to the database. AZ Roadmaps

Efficient Routing

Utilizing the power of Google's Directions API Arizona Roadmaps has fast navigation to any destination within the State of Arizona. Mark a location, search an address, or pick any of the AZ Roadmaps Campsites, and get efficient routing to your destination.
